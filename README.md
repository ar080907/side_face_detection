# Face Detection with OpenCV

A simple real-time face detection app using OpenCV and a Haar cascade classifier.

## Features
- Detects faces from webcam feed
- Displays bounding boxes and FPS on screen
- Adjustable detection size and scale factors
- The haarcascade is custom made

## Requirements
- Python 3.7+
- OpenCV (`pip install opencv-python`)

## Usage
1. Clone or download this repository
2. Place `cascade2.xml` in the same folder as `sidefacedetection.py`
3. Add the full directory of the cascade where mentioned
4. Run the program
